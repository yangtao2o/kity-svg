    toString: function() {
        return 'I am a person';

});

        return 'I am a man';

expand()(top, right, bottom, left)

Source:
  • graphic/box.js, line 234
扩展（或收缩）当前的盒子，返回新的盒子
Example
```js
var box = new kity.Box(10, 10, 20, 20);
var box1 = box.expand(10); // [0, 0, 40, 40]
var box2 = box.expand(10, 20); // [0, -10, 40, 60]
var box3 = box.expand(1, 2, 3, 4); // [9, 8, 24, 26]
```
Parameters:
Name Type Description
top Number 矩形区域的上边界往上扩展的值；如果是负数，则上边界往下收缩
right Number [Optional] 矩形区域的右边界往右拓展的值； 如果是负数，则右边界往左收缩； 如果不设置该值，使用和 top 同样的值。
bottom Number [Optional] 矩形区域的下边界往下拓展的值； 如果是负数，则下边界往上收缩； 如果不设置该值，使用和 top 同样的值。
left Number [Optional] 矩形区域的左边界往左拓展的值; 如果是负数，则左边界往右收缩; 如果不设置该值，使用和 right 同样的值。

paralle()(v1, v2, op)

Source:

平行地对 v1 和 v2 进行指定的操作 如果 v1 是数字，那么直接进行 op 操作 如果 v1 是对象，那么返回一个对象，其元素是 v1 和 v2 同键值的每个元素平行地进行 op 操作的结果 如果 v1 是数组，那么返回一个数组，其元素是 v1 和 v2 同索引的每个元素平行地进行 op 操作的结果
Example
```js
var a = {
    value1: 1,
    value2: 2,
    value3: [3, 4, 5]
};

var b = {
    value1: 2,
    value2: 3,
    value3: [4, 5, 6]
};

var c = kity.Utils.paralle(a, b, function(v1, v2) {
    return v1 + v2;
});

console.log(c.value1); // 3
console.log(c.value2); // 5
console.log(c.value3); // [7, 9, 11]

```
